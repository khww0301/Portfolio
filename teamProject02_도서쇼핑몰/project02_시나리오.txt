1. 상단 메뉴에서 회원가입 클릭 -> 회원정보 입력 -> 비밀번호 찾기 클릭 -> ID, PWD 입력으로 로그인 ->
2. 상단 메뉴에서 마이페이지 아이콘 클릭 -> 프로필 아이콘 클릭 -> 회원정보 수정 및 탈퇴 -> 
3. Test ID, PWD 입력으로 로그인 -> 메뉴에서 베스트 클릭 -> 체크 박스 선택 -> 장바구니 버튼 클릭 -> 
4. 장바구니 전체 삭제 클릭 -> 베스트셀러 페이지 주문하기 클릭 -> 주문/배송페이지에서 주문 확인 -> 
5. 메뉴에 있는 각 도서 클릭 -> 장르별 도서에서 서브 메뉴 클릭 -> 도서 클릭하여 도서 상세 페이지로 이동 -> 
6. 장바구니 담기 및 바로구매 클릭 -> 주문/배송으로 이동 -> 장바구니 아이콘 클릭 -> 
7. 체크 및 수량 등 선택 후 주문하기 클릭 -> 주문/배송으로 이동 -> 상단 메뉴에서 고객센터 클릭 -> 
8. 공지사항 더보기 클릭 -> 공지사항 쓰기 클릭 -> 공지사항 입력 후 write버튼 클릭 -> 공지사항 세부 항목 클릭 ->
9. 수정 버튼 클릭 -> 수정 내용 입력 후 write버튼 클릭 -> 상단 메뉴 고객센터 클릭 ->  1:1 문의 접수 아이콘 클릭 ->
10. 문의 내용 입력 후 접수 버튼 클릭 -> 문의 내용 세부 항목 클릭 -> 수정 버튼 클릭 ->
11. 수정 사항 입력 후 수정 버튼 클릭 -> 메인 페이지로 이동 -> 베스트셀러 소년이 온다 도서 클릭 ->
12. 도서 상세 페이지 하단 리뷰 작성 후 리뷰등록 클릭 -> 도서 리뷰 목록 수정 클릭 -> 
13. 수정 내용 입력 후 수정완료 클릭 -> 삭제 버튼 클릭
14. 로그아웃 클릭
